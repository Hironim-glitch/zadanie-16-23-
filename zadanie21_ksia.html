Użycie ID

Pojęcie Znaczenie pojęcia uzupełnia uczeń
Co znaczy skrót ID? Identifier (identyfikator).
Jaki jest cel stosowania ID? ID pozwala unikalnie zidentyfikować element na stronie.
Jakie jest znaczenie znaku #? Znak # jest używany do wywołania identyfikatora ID w CSS.
Jak następuje wywołanie ID (z wytłumaczeniem)? Wywołanie ID następuje za pomocą znaku # przed nazwą identyfikatora, np. #nazwa.
Uwagi na temat stosowania ID (trzy, konieczne wyliczenia):
  • ID musi być unikalny na stronie.
  • ID ma wyższy priorytet niż klasy.
  • ID można używać do łączenia z JavaScript.
