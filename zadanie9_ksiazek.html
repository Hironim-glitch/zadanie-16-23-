1. Składnia teoretyczna CSS

    2. Przykład składni CSS: 
    <style>
   
    body {
        background-color: #f0f0f0; 
        font-family: Arial, sans-serif;
    }
    p {
        color: blue; 
        font-size: 16px; 
    }
    h1 {
        color: green; 
        text-align: center;
    }
</style>

  2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować w dokumencie, jako podłączenie stylu CSS?

    3. Styl zewnętrzny jest umieszczany w sekcji <head> dokumentu HTML, a tag <link> jest odpowiedzialny za jego podłączenie. Można dodać tyle linków, ile potrzeba.

  3. Jak jest najprostsza forma wstawiania stylu zewnętrznego → otoczony ramką (może być tabela jedno komórkowa)?

  4. Ile arkuszy stylów może być dołączanych do jednego dokumentu HTML oraz jaka zasada musi być zachowana przy większej ilości dołączanych styli?

    5. Można dołączyć dowolną liczbę arkuszy stylów. Ważne jest, by każdemu arkuszowi przypisać oddzielny tag <link>.

  5. Co się dzieje w przypadku konfliktów? Przez konflikt rozumiemy, gdy te same style są definiowane w różnych arkuszach stylów podłączonych do tego samego dokumentu.

    6. W przypadku konfliktów, gdy style są zdefiniowane w różnych arkuszach, zastosowanie mają zasady kaskadowości. Ostatnia deklaracja w kodzie ma priorytet.

  6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    7. Pliki arkuszy są tworzone w prostym edytorze tekstowym, gdzie wpisuje się odpowiednią składnię CSS, a plik zapisuje z rozszerzeniem .css.

  7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    8. Specjalistyczny edytor CSS oferuje dodatkowe funkcje, takie jak podgląd na żywo i ułatwia proces tworzenia pliku. Plik zapisuje się automatycznie z rozszerzeniem .css.

  8. Jakie rozszerzenie posiada plik będący zewnętrznym arkuszem stylów?

    9. Zewnętrzny arkusz stylów zawsze posiada rozszerzenie .css.

  9. Jakie są zalety zewnętrznego arkusza stylów (dwie – z użyciem wyliczenia)?

    1. Wszystkie strony w obrębie serwisu mogą korzystać z tych samych, wspólnych cech.
    2. W przypadku zmiany np. czcionki na wszystkich stronach, wystarczy edytować tylko zewnętrzny arkusz stylów, co oszczędza czas w porównaniu do edytowania każdej strony osobno.

  10. Uwagi na temat pisania w CSS.

    1. Tag <link> należy umieścić wyłącznie w sekcji <head>.
    2. Można dołączyć dowolną liczbę arkuszy stylów.
    3. W przypadku konfliktów, później zadeklarowane style mają wyższy priorytet.
